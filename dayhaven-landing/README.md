# Dayhaven Landing Page

A premium, mobile-first landing page for [Dayhaven](https://dayhaven.app) — a parent school assistant web app.

## Live URL
https://dayhavenlp-sw7tqxy4.manus.space

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide React icons

## Design
- **Typography**: Fraunces (display serif) + Plus Jakarta Sans (body)
- **Palette**: Warm cream base (#FBF8F3), sage green, soft coral, muted amber, teal, charcoal
- **Style**: "Warm Stack" — editorial, magazine-like, color-blocked rounded cards
- **Mobile-first**: Designed for iPhone, scales beautifully to desktop

## Sections
1. Hero — Tagline, phone mockup, "Sign up free" CTA
2. Problem Statement — Scattered school communications
3. Feature Showcase — Dashboard, Calendar, Communications
4. How It Works — 3 simple steps
5. Pricing — Free tier + Pro at $39/year per household
6. Footer — CTA band + links

## Development
```bash
cd dayhaven-landing
pnpm install
pnpm dev
```
